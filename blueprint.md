# Guinea Pig Information Site

## Overview

A simple and visually appealing website dedicated to providing information about guinea pigs. The site will feature a main content section with details about guinea pigs, a header, and a footer. It will be styled with modern CSS and be mobile-responsive.

## Project Outline

### Style and Design

*   **Layout:** A single-column layout with a header, main content area, and footer.
*   **Colors:** A warm and inviting color palette will be used.
    *   Primary: `#8B4513` (SaddleBrown)
    *   Secondary: `#F5DEB3` (Wheat)
    *   Accent: `#228B22` (ForestGreen)
*   **Typography:** Clear and readable fonts.
    *   Headings: 'Georgia', serif
    *   Body: 'Arial', sans-serif
*   **Imagery:** A header image of a guinea pig.

### Features

*   **Header:** Contains the main title of the website.
*   **Main Content:**
    *   Introduction to guinea pigs.
    *   Sections on their diet, habitat, and care.
    *   A list of interesting facts.
*   **Footer:** Copyright information and links to resources.
*   **Interactivity:**
    *   A button that reveals a fun fact about guinea pigs.
